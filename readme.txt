== Description ==

This plugin places internal links in the tag descriptions to other tag pages. Your theme have to make use of <code>tag_description();</code> on the tag pages.

Visit the [Plugin Homepage](http://www.naden.de/blog/taxonomy-interlinker "Taxonomy Interlinker") for more details and help with the implementation of <code>tag_description();</code>. 

== Installation ==

1. Unpack the zipfile
1. Upload the folder `taxonomyinterlinker` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the plugin to fit your needs.
